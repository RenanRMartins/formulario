Link para o formulario
  1. O atributo "placeholder" é usado para mostrar um texto de exemplo dentro do campo de entrada, que desaparece quando o usuário começa a digitar.
  2. O atributo "checked" é usado para definir um botão de rádio ou caixa de seleção como selecionado por padrão.
  3. O atributo "name" deve ser o mesmo para todos os botões de rádio que você deseja agrupar.
  4. A tag label é usada para definir um rótulo para um elemento input. O atributo "for" associa o rótulo a um elemento input específico, usando o valor do atributo "id" do input.
  5. O atributo "selected" define uma opção como selecionada por padrão em um menu suspenso.
  6. O atributo "required" torna um campo obrigatório, impedindo o envio do formulário até que o campo seja preenchido.
  7. A tag textarea é usada para criar uma área de texto multilinha, permitindo que o usuário insira texto longo.
  8. O atributo "action" especifica a URL para a qual os dados do formulário serão enviados quando o formulário for submetido.
  9. O atributo "method" especifica o método HTTP a ser usado ao enviar os dados do formulário (geralmente "GET" ou "POST").
  10. O valor deve ser "POST".
  11. O valor deve ser "POST", e o atributo "enctype" deve ser "multipart/form-data".
  12. As tags fieldset e legend são usadas para agrupar elementos de um formulário e fornecer uma legenda para o grupo, respectivamente.
  13. As tags HTML fieldset e legend são usadas para agrupar elementos de formulário, a tag fieldset cria uma área com borda ao redor de campos de formulário, organizando visualmente campos relacionados ja a tag legend fornece um título ou descrição para o grupo de campos, que aparece no topo do fieldset.
